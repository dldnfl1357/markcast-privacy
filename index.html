MarkCast는 사용자의 개인정보와 문서를 최소한으로 처리하도록 설계된 마크다운 음성 리더입니다.

수집하거나 공유하는 데이터

MarkCast 개발자는 개인정보, 사용 기록, 기기 식별자, 위치, 진단 정보 또는 문서 내용을 수집하거나 제3자와 공유하지 않습니다. 앱에는 계정, 광고, 분석 SDK, 추적 기술과 개발자 운영 백엔드 서버가 없습니다.

사용자가 선택한 문서

앱은 시스템 파일 선택기를 통해 사용자가 직접 선택한 파일에만 접근합니다. 문서 내용은 화면 표시와 음성 읽기를 위해 실행 중 메모리에서 처리되며 개발자 서버로 업로드하거나 영구 저장하지 않습니다.

시스템 음성 합성

MarkCast는 별도의 클라우드 음성 서비스를 사용하지 않고 Android 또는 iOS의 시스템 TTS API에 읽을 텍스트를 전달합니다. 사용자가 선택한 운영체제 음성 엔진이나 네트워크 음성은 해당 제공자의 정책에 따라 처리할 수 있으며, MarkCast 개발자는 그 텍스트나 결과를 수신하지 않습니다. 완전한 오프라인 처리를 원하면 기기 설정에서 오프라인 음성을 내려받아 선택할 수 있습니다.

권한, 보관과 삭제

Android 앱은 인터넷 권한이나 광범위한 사진·미디어·저장소 권한을 요청하지 않으며 원격 마크다운 이미지를 자동 다운로드하지 않습니다. 개발자가 수집·보관하는 서버 데이터는 없습니다. 다른 문서를 열거나 앱을 완전히 종료하면 현재 문서를 메모리에서 제거할 수 있습니다.

아동, 변경 및 문의

앱은 아동을 포함한 누구의 개인정보도 의도적으로 수집하지 않습니다. 처리 방식이 바뀌면 이 정책과 시행일을 갱신합니다. 문의는 MarkCast 지원 페이지를 이용해 주세요.

English

MarkCast is a Markdown text-to-speech reader designed to minimize the handling of personal information and document content.

Data collected or shared

The developer does not collect or share personal information, usage history, device identifiers, location, diagnostics, or document content. The app has no accounts, advertising, analytics SDKs, tracking technology, or developer-operated backend.

User-selected documents

The app accesses only files explicitly selected through the system file picker. Content is processed in runtime memory for display and speech; it is not uploaded to a developer server or permanently stored.

System text-to-speech

MarkCast uses the Android or iOS system TTS API instead of a separate cloud speech service. A system speech engine or network voice selected by the user may process text under its provider's policy. The MarkCast developer does not receive the text or result. Users who require fully offline processing can download and select an offline voice in device settings.

Permissions, retention, and deletion

The Android app requests neither internet access nor broad photo, media, or storage permissions, and remote Markdown images are not downloaded. The developer retains no server data. Opening another file or fully closing the app removes the current document from memory.

Children, changes, and contact

The app intentionally collects no personal data from anyone, including children. If its practices change, this policy and effective date will be updated. Contact the developer through the MarkCast support page.
