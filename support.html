빠른 사용법

  1. 앱에서 마크다운 파일 열기를 누르고 .md, .markdown 또는 .txt 문서를 선택합니다.
  2. 하단 플레이어에서 언어와 속도를 선택한 뒤 읽기 시작을 누릅니다.
  3. 외부 파일 없이 확인하려면 시작 화면의 샘플 문서로 둘러보기를 이용합니다.

문제 해결

  • 음성이 들리지 않으면 기기 음량과 무음 모드를 확인하고, 시스템 설정에서 필요한 언어의 TTS 음성을 내려받습니다.
  • 파일이 열리지 않으면 파일 크기가 5MB 이하인지, UTF-8 또는 UTF-16 텍스트 문서인지 확인합니다.
  • 원격 마크다운 이미지는 개인정보 보호를 위해 자동으로 내려받지 않습니다.

문의

오류 제보나 지원 요청은 MarkCast 공개 지원 게시판에 남겨 주세요. 문서 원문이나 개인정보는 첨부하지 마세요.

Quick start

  1. Tap Open Markdown file and select a .md, .markdown, or .txt document.
  2. Choose a language and speed in the bottom player, then tap Start reading.
  3. Use Explore with a sample document to test the app without an external file.

Troubleshooting

  • If speech is silent, check device volume and silent mode, then download the required system TTS voice in device settings.
  • If a file will not open, confirm it is no larger than 5MB and is UTF-8 or UTF-16 text.
  • Remote Markdown images are not downloaded automatically to protect privacy.

Contact

Submit bug reports or support requests on the public MarkCast support board. Do not attach private documents or personal information.
